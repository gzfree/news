Video Call Enhancement Features

ü™ü Overview

As video calling frequency continues to rise, users increasingly demand visual personalization, privacy protection, and interactive entertainment features. Botim's current video calling experience is relatively basic, lacking mainstream features like background replacement, beauty filters, virtual avatars and other personalization options that users need in social, entertainment, and privacy-focused scenarios.

This project aims to enhance Botim's VoIP video calls by implementing visual enhancement features that will improve user experience, increase product differentiation, and drive monetization through membership-exclusive visual assets:

  • Background Replacement: Users can protect their privacy by blurring or replacing their background during video calls (requires image segmentation SDK).
  • Beauty Filters: Users can enhance their appearance with skin smoothing, whitening, face slimming and other adjustable effects (requires video SDK).
  • Video Filters: Users can apply various style filters like cool/warm tones, vintage effects, etc. (requires GPU rendering).
  • Virtual Avatars: Users can replace their real appearance with virtual characters during calls (requires Avatar SDK).
  • Avatar Accessories: Users can add fun decorative elements to their appearance (frontend implementation).
  • Premium Visual Benefits: Certain filters/avatars/accessories available exclusively to members (requires membership system integration).

üõ∏ Problem

  • Users need more engaging features to increase video call duration, directly impacting advertising revenue potential.
  • Current VoIP solutions lack compelling reasons for users to upgrade to premium memberships.
  • User engagement metrics show declining interest in standard video calls without interactive elements.
  • Competitors are introducing similar features, creating market pressure to innovate.

üéØ Objectives

  • Improve video call user experience and differentiate Botim from competitors by implementing visual enhancement features.
  • Increase video call usage duration by 10%.
  • Achieve 10% AR feature usage rate among all active users.
  • Achieve ‚â•50% usage rate of visual features among members in the month following subscription.
  • Increase non-member conversion rate by ‚â•3% by using trial features as conversion entry points.
  • Prioritize background replacement and beauty features (P0) for initial release, followed by filters and avatars (P1), with accessories and premium benefits in later phases (P2).

üöß Constraints

  • Only available for video calls; voice calls do not support this feature.
  • AI processing must be done on-device for privacy and low latency.
